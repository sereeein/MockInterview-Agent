# 小红书 Week 3：前端 + 雷达图报告

**标题**：我用 4 周做一个 AI 面试 Agent #3：完整 UI 跑起来了

**正文**：

第 3 周前端完工。Next.js 16 + shadcn/ui + Recharts。

8 个核心页面：
- 上传页（拖入 PDF + 选岗位 + 粘 JD）
- 题库页（12 道题分 5 类，状态 / 最高分 / 上次练习时间）
- 单题演练（聊天 UI，4 个快捷按钮：跳过 / 换场景 / 求提示 / 结束）
- 单题报告（雷达图 + rubric 4 维度评分 + 改进建议 + 范例答案 + 完整对话）
- 模拟面试（5 题串联）+ 整套报告（柱状图 + 高光 / 短板 / 下一步建议）

最满意的细节：报告里的「场景切换」轮次有 amber ring + 「思考框架」轮次有 blue ring，看 transcript 一眼能看出 agent 给过几次台阶 / 用户求过几次提示。

下周收尾：评估 + 部署 + 写简历金句。
